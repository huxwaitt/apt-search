Grab Apartments.com listings → apartments_listings.json

One-time setup for the apartment-search skill's Apartments.com source. Apartments.com 403s automated scrapers, so the listings have to come from your own browser — this bookmarklet reads the results page you're looking at. No automation touches it.

Setup (once)

  1. Make sure your bookmarks bar is visible (Ctrl+Shift+B in Chrome/Edge, Cmd+Shift+B on Mac).
  2. Drag this button up onto your bookmarks bar:
    🏢 Grab Apartments.com
Can't drag it? Create a new bookmark manually, name it Grab Apartments.com, and paste the code below as its URL:

Each time you search

  1. Open an Apartments.com search for your city/filters, e.g. https://www.apartments.com/falls-church-va/2-bedrooms-under-2500/. Clear any check, and scroll down once so photos load.
  2. Click the Grab Apartments.com bookmark. An alert shows the count and apartments_listings.json downloads to your Downloads folder.
  3. Tell Claude it's ready — the search picks up the newest dump automatically (--apartments-json).
