Grab Zillow listings → zillow_listings.json

One-time setup for the apartment-search skill's Zillow fallback. No automation touches your browser, so Zillow's "Press & Hold" check never trips.

Setup (once)

  1. Make sure your bookmarks bar is visible (Ctrl+Shift+B in Chrome/Edge).
  2. Drag this button up onto your bookmarks bar:
    📋 Grab Zillow listings
Can't drag it? Create a new bookmark manually, name it Grab Zillow listings, and paste the code below as its URL:

Each time you search

  1. Open the rentals page for your city, e.g. https://www.zillow.com/falls-church-va/rentals/. Clear any "Press & Hold" check so listings show.
  2. Click the Grab Zillow listings bookmark. An alert shows the count and zillow_listings.json downloads to your Downloads folder.
  3. Tell Claude it's ready — the search picks up the newest dump automatically (--zillow-json).
